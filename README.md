## Description

[tsul](https://github.com/wlsdud2194/tsul) is an open-source project that provides frequently used and useful methods for development. (beta)

## Installation
```
npm install tsul

yarn add tsul
```

- utility
  - [Array Utility](src/utility/arrayUtil.ts)
